# Student Mental Health Analysis

## Overview

This project explores major factors that may influence student mental health using survey data. The analysis investigates relationships between academic pressure, sleep, study satisfaction, financial concerns, and mental wellbeing.

## Objectives

- Explore the student mental health dataset.
- Clean and prepare the data.
- Create visualisations to identify patterns.
- Summarise key findings from the analysis.

## Dataset

The dataset contains survey responses from university students and includes variables such as:

- Gender
- Age
- CGPA
- Academic Pressure
- Average Sleep
- Study Satisfaction
- Financial Concerns
- Anxiety
- Depression
- Future Insecurity

## Technologies Used

- Python

- Matplotlib

## Key Findings

The exploratory analysis suggests that factors such as academic pressure, sleep duration, study satisfaction, and financial concerns may be associated with student mental health outcomes.

These findings represent patterns in the data and should not be interpreted as evidence of causation.
